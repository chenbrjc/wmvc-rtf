# This app has been created with NetBeans IDE, GlassFish application server, and PostgreSQL database.

### Prerequisites: ###

1.  Download and install <a href="https://netbeans.org/">NetBeans IDE</a> (e.g., Version 8.x).
2.  Download and install <a href="https://www.postgresql.org/">PostgreSQL</a> (e.g., Version 9.5.x).

### Getting started with the app: ###

1.  Open PostgreSQL admin console and excute the scripts in PostgreSQL.txt.
2.  Start NetBeans IDE and open dwmvc-angular-java-sql project.
3.  Select and run dwmvc-angular-java-sql project.
5.  Point a browser to: http://localhost:8080/wmvcapp.
6.  Point another browser to the same URL.
7.  Text entries in one browser will reflect in another right away.
